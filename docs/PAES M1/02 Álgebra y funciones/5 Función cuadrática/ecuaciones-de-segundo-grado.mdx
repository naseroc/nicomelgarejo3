## Resolver ecuaciones cuadr√°ticas factorizando

Cuando el lado izquierdo de la ecuaci√≥n $ax^2 + bx + c = 0$ es factorizable, puedes resolver la ecuaci√≥n usando la propiedad del producto cero. Este m√©todo se debe usar cuando se puedan factorizar f√°cilmente las ecuaciones cuadr√°ticas.

<div className="definition-box">
<strong>Propiedad del producto cero</strong>  <br/>

Algebraicamente, si $P$ y $Q$ son expresiones y $PQ=0$, entonces $P = 0$ o $Q = 0$.
</div>

Por ejemplo, para encontrar las soluciones de $x^2-4x+3=0$ puedes factorizar $x^2-4x+3$ como $(x-3)(x-1)$, obteniendo la ecuaci√≥n equivalente $(x-3)(x-1)=0$. Aplicando la propiedad del producto cero, se sabe que $x-3=0$ o $x-1=0$, concluyendo que $x=3$ o $x=1$.

## Resolver ecuaciones cuadr√°ticas calculando ra√≠ces cuadradas

Este m√©todo se debe usar cuando la ecuaci√≥n puede ser escrita de la forma $u^2 = d$, donde $u$ es una expresi√≥n algebraica. Al escribir la ecuaci√≥n en la forma $u^2 = d$, √©sta se resuelve sacando la ra√≠z cuadrada de cada lado de la igualdad.

Otro ejemplo es resolver la ecuaci√≥n $4x^2-31=49$. Al sumar 31 a cada lado de la igualdad, se obtiene $4x^2=80$. Luego, dividiendo por 4 a ambos lados, se obtiene $x^2=20$. Finalmente, se concluye que las soluciones son $x=\sqrt{20}$ y $x=-\sqrt{20}$.  

Las soluciones de $x^2-16x + 64 = 100$ tambi√©n pueden ser encontradas empleando este m√©todo, puesto que $x^2-16x + 64$ es igual que $(x-8)^2$. As√≠, $x^2-16x + 64 = 100$ equivale a $(x-8)^2 = 100$. Al calcular la ra√≠z cuadrada a ambos lados de la igualdad, $\sqrt{(x-8)^2}=|x-8|$ y $\sqrt{100}=10$. De la igualdad $|x-8|=10$ se concluye que $x=18$ y $x=-2$.

## Resolver ecuaciones cuadr√°ticas completando cuadrados

En el ejemplo anterior, la expresi√≥n $x^2-16x + 64$ es un trinomio cuadrado perfecto porque es igual al cuadrado de binomio $(x-8)^2$.  

<div className="definition-box">
<strong>Importante:</strong> A veces necesitas sumar un t√©rmino a una expresi√≥n $x^2+bx$ para convertirla en un trinomio cuadrado perfecto. Este proceso se llama *completar el cuadrado*.  
Para completar el cuadrado de la expresi√≥n $x^2+bx$, debes sumar $\left(\frac{b}{2}\right)^2$ como se muestra a continuaci√≥n:  
$$
\begin{aligned}
x^2 + bx + \left(\frac{b}{2}\right)^2
&= \left(x + \frac{b}{2}\right)\left(x + \frac{b}{2}\right) \\
&= \left(x + \frac{b}{2}\right)^2
\end{aligned}
$$
</div>

Por ejemplo, para encontrar el valor de $c$ que hace que $x^2 + 14x + c$ sea un trinomio cuadrado perfecto, debes calcular la mitad de 14 y elevar el resultado al cuadrado. As√≠, $c=\left(\dfrac{14}{2}\right)^2=7^2=49$. Entonces, $x^2 + 14x + 49 = (x + 7)(x + 7) = (x + 7)^2$.  

<div className="definition-box">
<strong>Importante:</strong> El m√©todo de completar el cuadrado puede usarse para cualquier ecuaci√≥n cuadr√°tica $ax^2 + bx + c = 0$, pero es m√°s f√°cil aplicarlo cuando $a = 1$ y $b$ es un n√∫mero par. Cuando completes el cuadrado como parte de resolver una ecuaci√≥n, deber√°s sumar el mismo n√∫mero a ambos lados de la ecuaci√≥n.  
</div>

Por ejemplo, al resolver $x^2-10x + 7 = 0 $ completando el cuadrado, reescribes la ecuaci√≥n restando 7 a ambos lados de la igualdad: $x^2-10x=-7$. Sumas a ambos lados $\left(\dfrac{-10}{2}\right)^2=25$ y obtienes $x^2-10x+25=-7+25$. Con esto puedes reescribir la igualdad como $(x-5)^2=18$ y aplicando ra√≠z cuadrada a ambos lados de la igualdad se obtiene que $x=5+\sqrt{18}$ y $x=5-\sqrt{18}$.  

## Resolver ecuaciones cuadr√°ticas usando la f√≥rmula cuadr√°tica

Para cualquier ecuaci√≥n cuadr√°tica, al completar el cuadrado para la ecuaci√≥n general $ax^2 + bx + c = 0$ se obtiene la f√≥rmula para obtener las soluciones de la ecuaci√≥n. Esta f√≥rmula se denomina **f√≥rmula cuadr√°tica**.

$$
\begin{aligned}

    &= \dfrac{4\pm \sqrt{16-12}}{2} \\
    &= \dfrac{4\pm \sqrt{4}}{2} \\

\end{aligned}
$$

Entonces, las soluciones de la ecuaci√≥n $x^2-4x+3=0$ son $x=\dfrac{4+2}{2}=\dfrac{6}{2}=3$ y $x=\dfrac{4-2}{2}=\dfrac{2}{2}=1$.

<div className="definition-box">
<strong>Importante:</strong> En la f√≥rmula cuadr√°tica, la expresi√≥n $b^2-4ac$ se llama el **discriminante** de la ecuaci√≥n asociada $ax^2 + bx + c = 0$.  <br/>
$\star$ Si el discriminante es mayor que cero, la ecuaci√≥n tiene dos soluciones distintas y reales.  <br/>
$\star$ Si el discriminante es igual que cero, la ecuaci√≥n tiene dos soluciones iguales y reales (una soluci√≥n).  <br/>
$\star$ Si el discriminante es menor que cero, la ecuaci√≥n tiene dos soluciones que no son reales.
</div>
